Telegram-аналитик для видеофайлов с использованием современных технологий обработки естественного языка, промпт-инжиниринга и искусственного интеллекта. Позволяет пользователям загружать видео и получать детальную аналитику в автоматическом режиме.
Бот имеет гибридный, многоуровневый функционал, базовые команды работают напрямую через SQL-запросы, сложные запросы обрабатываются гигачат-LLM и выдают сложную аналитику на основе промптирования.
Производится охват всего временного периода и всех метрик согласно JSON-файла,в том числе производится анализ отсутсвия информации за те периоды, которые не охватывает JSON.

Основной функционал:

Автоматический анализ видеофайлов через телеграм-интерфейс.

Генерация текстовых отчетов с помощью SQL-запросов и AI (GigaChatAI).

Извлечение ключевых кадров и метаданных.

Хранение истории анализов в базе данных.

Интуитивное, разговорное и командное общение.

Программа имеет объемное тестовое покрытие, состоящее из Unit-тестов (модульных), интеграционных, функциональных, тестов граничных случаев, тестов обработки ошибок, параметризованных тестов,
тестов конфигурации, тестов безопасности/SQL инъекций, тестов форматов и парсинга, тестов логирования, тестов кэширования, тестов состояния (State tests), покрывающих 89% всего функционала.

Запуск проекта:
Подразумевает у пользователя наличие Docker, PostgreSQL в различных интерфесах, наличие Gigachat2 модели lite.

Необходимая информация для интеграции: (src/.env) Проект уже настроен на работу с DB_HOST=localhost, DB_PORT=5432, DB_NAME=video_stats, для активации LLM в полях GIGACHAT_CLIENT_ID= и GIGACHAT_CLIENT_SECRET= ввести свои данные.

Создание виртуального окружения

Установка всех зависимостей: pip install -r requirements.txt

Инициализация и создание таблиц в БД, в "человеческом формате ID" (JSON имеет только UUID-формат): db>python init_db.py

Запуск программы из точки входа: >cd src, >python app.py (логирование покажет подробную информацию о подключении всего функционала)

Запуск тестирования: базовая директория>python run_tests.py и в конце программа выдаст результат в процентном соотношении в виде таблицы.

Проект полностью готов к работе.
